## Распределение ip адресации и интерфейсов

ip адресация распределена по правилам.

Lo SPINE = 10.0.0.№Spine \
Lo LEAF = 10.0.255.№Leaf \
P2P /31 = 10.0.№Spine.X, Где X-порядковок 0-255, от 1Leaf до NLeaf 

Интрефейсы: \
Для SPINE, xe-0/0/№Leaf \
Для Leaf, xe-0/0/№Spine
